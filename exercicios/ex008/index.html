Principais Formatações

Negrito | Destaque

Nesta frase temos um termo em negrito usando a tag <b> (Não Semântica)

Nesta frase temos um termo em destaque usando a tag < strong> (Semântica)

Itálico | Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (Não Semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (Semântica)

Texto Marcado

Podemos criar um texto marcado usando a tag <mark>

Texto Grande e Pequeno

Estamos criando um texto pequeno nesse parágrafo.

Texto Excluído | Inserido

Podemos marcar um texto excluído para indicar que ele não deve ser considerado

Também podemos marcar um texto inserido para indicar que ele foi adicionado usando a tag <ins>. (Semântica)

E por fim podemos marcar um texto inserido para indicar que ele foi adicionado usando a tag <u>. (Não Semântica)

Texto Sobrescrito | Subscrito

Para inserir fórmulas matemáticas do tipo x20+3 usando a tag <sup>.

Também para inserir fórmulas químicas do tipo H20 usando a tag <sub>.
